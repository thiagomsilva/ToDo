# ToDo APP

Aplicação To Do List utilizando bootstrap

## Gems utilizadas

- Ruby 3.1.2
- Rails 6.1.7
- Bootstrap 5.2.3
- PostgreSQL
- RSpec
- Factory bot
- Faker
- Shoulda matchers

## Inciando a aplicação

- Intale as gems:

```
bundle install
```

- Criando o banco de dados e rodando as migrations:

```
rails db:create db:migrate
```

- Executar os testes:

```
rspec
```

- Inicar o servidor local:

```
rails s
```

- Acesse no navegador:

http://localhost:3000
